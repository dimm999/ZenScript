ZenScript
Where Focus Meets Aesthetics.
ZenScript is a beautiful, minimalist, block-based text editor designed to help you enter a deep state of flow. Inspired by the elegance of tools like Zen Mode, Scrivener, and Notion, it combines powerful document organization with a soothing, distraction-free writing environment.
Whether you are drafting a novel, organizing daily notes, or structuring a complex project, ZenScript gets out of your way so your ideas can take center stage.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
